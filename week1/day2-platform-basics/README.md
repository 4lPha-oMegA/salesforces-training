1\. What is the Salesforce Platform?

The Salesforce Platform is a cloud-based framework designed for building and managing business applications. It operates on a Multi-tenant Architecture, meaning multiple organizations share the same underlying infrastructure and resources while keeping their data completely isolated and secure.

============================================================================================================================================

2\. Core Components: App, Object, and Tab:

App: A collection of tabs and objects that work together to serve a specific business function (e.g., a "Sales" app).



Object: A database table where records are stored. For example, the "Contact" object holds information about people.   



Tab: A visual navigation element that acts as a starting point or "window" to view an object's records.



============================================================================================================================================      



3\. Difference Configuration Between Coding



Salesforce allows developers to extend functionality using two main approaches:



Configuration:



1. Approach Use Case Example Configuration Use for standard tasks, faster and easier to maintain.

2\. Creating a custom "Loan Amount" field or setting up a validation rule.   



Coding:



1. Coding (Apex)Use for complex logic or external system integrations.   

2\. Building a custom API to connect to an external bank or creating a complex multi-step UI.
